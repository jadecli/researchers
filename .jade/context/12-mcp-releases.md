# Model Context Protocol Specification Releases

## Release 2025-11-25

- **Tag**: v2025-11-25
- **Commit**: specification update
- **Creator**: MCP team
- **Notes**: Latest specification release. Includes streamable HTTP transport, tool annotations, elicitation, structured output for tools, and audio content type support.

## Release 2025-08-01

- **Tag**: v2025-08-01
- **Commit**: specification update
- **Creator**: MCP team
- **Notes**: Added OAuth 2.1 authentication framework, resource templates with URI templates, completion support for arguments, and improved error handling specification.

## Release 2025-06-18

- **Tag**: v2025-06-18
- **Commit**: specification update
- **Creator**: MCP team
- **Notes**: Added pagination support for list operations, middleware specification, and logging level configuration. Refined tool result content types.

## Release 2025-03-26

- **Tag**: v2025-03-26
- **Commit**: specification update
- **Creator**: MCP team
- **Notes**: Added sampling capability for server-initiated LLM calls, notification types for progress tracking, and root list change notifications.

## Release 2024-12-15

- **Tag**: v2024-12-15
- **Commit**: specification update
- **Creator**: MCP team
- **Notes**: Refined transport specifications. Added SSE transport alongside stdio. Improved capability negotiation between clients and servers.

## Release 2024-11-05

- **Tag**: v2024-11-05
- **Commit**: specification update
- **Creator**: MCP team
- **Notes**: First public specification release. Defined core protocol: JSON-RPC 2.0 messaging, tools, resources, prompts, stdio transport, capability negotiation.

## Release 2024-10-07

- **Tag**: v2024-10-07
- **Commit**: initial release
- **Creator**: MCP team
- **Notes**: Initial draft specification. Established the Model Context Protocol as an open standard for connecting AI models to external data sources and tools.
