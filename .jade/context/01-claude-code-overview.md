# Claude Code Overview

Claude Code is an agentic coding tool that lives in your terminal, IDE, desktop, and web — letting you build software through natural conversation. It understands your entire codebase, edits files, runs commands, searches the web, and manages complex engineering workflows — all without switching context.

## Installation

### Terminal (native install)

```bash
npm install -g @anthropic-ai/claude-code
```

### Terminal (Homebrew)

```bash
brew install claude-code
```

### Terminal (WinGet)

```bash
winget install Anthropic.ClaudeCode
```

### VS Code

Install the Claude Code extension from the VS Code Marketplace.

### Desktop app

Download the Claude Code Desktop app from code.claude.com.

### Web

Use Claude Code directly in your browser at code.claude.com.

### JetBrains

Install the Claude Code plugin from the JetBrains Marketplace.

## What you can do

### Automate work

Automate repetitive tasks, refactor code, and handle complex engineering workflows through natural language instructions.

### Build features

Describe what you want to build and Claude Code will implement it — creating files, writing code, and wiring everything together.

### Create commits

Generate well-structured git commits with meaningful messages based on the changes you've made.

### MCP (Model Context Protocol)

Connect Claude Code to external tools and data sources using MCP servers for enhanced capabilities.

### Customize

Tailor Claude Code to your workflow with CLAUDE.md files, custom slash commands, and hooks.

### Agent teams

Coordinate multiple Claude Code agents working together on different parts of your project using the Agent SDK.

### Pipe / Script / Automate

Use Claude Code in scripts and pipelines with `claude -p` for non-interactive mode and programmatic access.

### Schedule tasks

Schedule Claude Code to run tasks automatically on a recurring basis.

### Work from anywhere

Access Claude Code from your terminal, IDE, desktop app, or web browser — your workflow follows you.

## Use Claude Code everywhere

| Environment | How to access | Best for |
|---|---|---|
| Terminal | `npm install -g @anthropic-ai/claude-code` then run `claude` | Full-featured agentic coding, CI/CD, scripts |
| VS Code | Install extension from Marketplace | IDE-integrated coding assistance |
| JetBrains | Install plugin from Marketplace | IDE-integrated coding assistance |
| Desktop | Download from code.claude.com | Standalone desktop experience |
| Web | Visit code.claude.com | Browser-based access, no install needed |
